## 詳細ページ

一覧の画像を押すと、各データの詳細ページへ遷移できます。

## Memoria 詳細

- 同名メモリアのカード種別違いをタブで切り替えできます。
- レギオンマッチスキル、補助スキル、レジェンダリースキル、対ヒュージスキルを確認できます。
- レベルごとのステータス表を表示します。

## Costume 詳細

- レアスキルの説明と補足を確認できます。
- `EX` / `ADX` スキルの内容を確認できます。
- `ADX` は突破数と覚醒状態を切り替えて確認できます。
- ステータス表では共通値、固有値、ジョブ補正を確認できます。

## Character 詳細

- キャラクターの基本情報、所属、紹介文を表示します。
- 対応する衣装一覧をそのまま辿れます。
- 前後のキャラクターへ移動できます。
