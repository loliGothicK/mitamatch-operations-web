## Data Pages とは

Data Pages は、メモリア、衣装、キャラクターを検索・参照するためのページ群です。
グリッドで一覧を見ながら、クエリで絞り込みや並び替えを行い、詳細ページまで遷移できます。

## 主な機能

| Feature                         | Description |
|---------------------------------|-------------|
| [Query](/docs/data/query)       | GoogleSQL ライクなクエリで一覧を絞り込みます。 |
| [Detail](/docs/data/detail)     | 各データの詳細ページでスキルやステータスを確認します。 |

## 対応しているタブ

- `Memoria`
  メモリア一覧と詳細ページを表示します。
- `Costume`
  衣装一覧と詳細ページを表示します。
- `Character`
  キャラクター一覧と、そのキャラクターに紐づく衣装一覧を表示します。

## 使い方

1. [`/data`](/data) を開き、見たいタブを選択します。
2. クエリエディタで条件を入力して一覧を絞り込みます。
3. 一覧の画像や名前から詳細ページへ遷移します。
4. 必要に応じて共有リンクを生成します。
