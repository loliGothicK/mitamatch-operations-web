- 左カラム
  現在のタイムラインです。オーダーの発動順、準備時間、終了時刻を確認できます。
- 右カラム
  追加可能なオーダー一覧です。カテゴリフィルタと課金/無課金の切り替えに対応しています。
- 上部操作
  Share、カテゴリフィルタ、課金/無課金切り替えを操作します。

## タイムラインの作成

- オーダー一覧の `+` を押すとタイムラインへ追加されます。
- 同じカテゴリのオーダーは複数追加できません。
- タイムライン左端のハンドルをドラッグすると並び替えできます。
- 各行には準備開始時刻、準備時間、終了時刻が表示されます。
- 編集ボタンから `delay`、`PIC`、`Sub PIC` を変更できます。
- 共有リンクは short link / full link の 2 種類を生成できます。
