## Timeline Builder とは

Timeline Builder では、レギオンマッチ用のオーダーを時系列で組み立てられます。
オーダーの追加、並び替え、担当者入力、ディレイ調整、共有リンク生成に対応しています。

## 使い方

1. 右側のオーダー一覧から `+` を押してタイムラインに追加します。
2. 左側のタイムラインでドラッグして順番を調整します。
3. 編集ボタンから `delay`、`PIC`、`Sub PIC` を更新します。
4. Share メニューから short link / full link を生成して共有します。
