# Builder

## 画面構成

![structure](/docs/timeline-builder.png)

- 基本

- 1. タイムラインが表示される場所
- 2. オーダーリスト

- ボタン

- 3. 共有リンクを生成するボタン
- 4. オーダーリストを切り替えるドロップダウン
- 5. 課金オーダーと無課金オーダーを切り替えるボタン

## タイムラインの作成

- オーダーリストの画像左上にある「+」ボタンをクリックすると、タイムラインに追加される
- タイムラインのオーダーは左端のインジケータでドラッグして並び替えることができる
