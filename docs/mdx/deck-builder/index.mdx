# Deck Builder

## Deck Builder とは

デッキビルダーは、効率的にユニットを編成するためのツールです。

デッキビルダーを使用することで、デッキを作成・編集・共有することができます。
ユニットのステータスを計算したり、メモリアを入れ替えたときにどのような変化があるかを確認することができます。

## 主な機能

デッキビルダーのメイン機能はつぎの機能を提供しています:

| Feature                                     | Description             |
|---------------------------------------------|-------------------------|
| [Builder](/docs/deck-builder/builder)       | デッキを作成・編集・共有することができます。  |
| [Calculator](/docs/deck-builder/calculator) | ユニットのステータスを計算することができます。 |

## 使い方

デッキビルダーを効果的に使用するには、つぎの手順を実行してください。

1. デッキを作成する
2. 実際にレギオンマッチを行い、録画やバトルログを解析する
3. 足りないバフやデバフを確認し、デッキを改善する
