# Welcome to Mitamatch Operations!

Mitamatch Operations へようこそ！
このドキュメントは Mitamatch Operations の基本的な使い方を説明します。

## What is Mitamatch Operations?

Mitamatch Operationsは、アサルトリリィ Last Bullet（ラスバレ） におけるレギオンマッチのための強力なサポートツールです。
このツールは無料で、広告も表示されません。

Mitamatch Operationsの主な機能は、ユニット編成の最適化を支援することです。
これにより、ユニットの編成にかかる時間を削減し、戦略の立案により多くの時間を割くことができます。
もちろんオーダーのタイムラインを作成することもできます。

Mitamatch Operationsは、経験豊富なレギオンの一員から初心者まで、
レギオンマッチを楽しむすべてのプレイヤーに対して、インタラクティブでダイナミックなレギオンマッチの構築を高速に支援します。

## Main Features

以下の機能を提供しています:

| Feature                                    | Description                     |
|--------------------------------------------|---------------------------------|
| [Deck Builder](/docs/deck-builder)         | ユニットを作成・共有することができます。            |
| [Timeline Builder](/docs/timeline-builder) | オーダーのタイムラインを作成・共有することができます。     |
| [Flow Chart](/docs/flow-chart)             | 複雑なオーダーの動きをフローチャートで表現することができます。 |
| [Database](/docs/database)                 | メモリアの実装状況を確認できます。               |
