## Deck Builder とは

Deck Builder は、ユニット編成を組み立てながら期待値や構成差分を確認するためのツールです。
メモリアの追加・並び替え・濃度変更・比較・共有を 1 画面で行えます。

## 主な機能

デッキビルダーのメイン機能はつぎの機能を提供しています:

| Feature                                     | Description                                        |
| ------------------------------------------- | -------------------------------------------------- |
| [Builder](/docs/deck-builder/builder)       | デッキの作成、比較、共有、絞り込み操作を行います。 |
| [Calculator](/docs/deck-builder/calculator) | 現在の編成をもとに期待値や各種設定を確認します。   |

## 使い方

デッキビルダーを効果的に使用するには、つぎの手順を実行してください。

1. Builder タブでデッキを組む
2. Filter / Search / Sort で候補を絞る
3. Calculator タブで期待値を確認する
4. 必要に応じて共有リンクを生成する
