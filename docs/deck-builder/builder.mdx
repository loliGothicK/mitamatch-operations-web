- 左カラム
  現在のユニット情報、ステータス、スキル詳細を表示します。
- 中央カラム
  デッキ本体です。メモリアの追加、削除、並び替え、比較対象の確認を行います。
- 右カラム
  メモリア一覧です。フィルタや検索条件に応じた候補が表示されます。
- 上部ツールバー
  デッキのクリア、比較、共有、計算設定、Filter、Search、Sort を操作します。

## ユニットの作成

- 右側のメモリア一覧で `+` を押すとデッキに追加されます。
- デッキ内のメモリアはドラッグで並び替えできます。
- `Filter` では役割、属性、ラベル、所持メモリアで候補を絞り込めます。
- `Search` ではステータス上昇/下降、特殊効果、支援効果などを条件に検索できます。
- `Sort` では候補一覧の並び順を切り替えできます。
- `Share` では short link / full link を発行できます。
- 比較モードを使うと、差し替え前後の期待値差分を確認できます。
