## Introduction

Calculator は現在のデッキをもとに、ダメージ・バフ・デバフ・回復の期待値を確認するためのタブです。
Charm、Costume、敵防御値、自己ステータス、AD レベルなどの条件を変えながら結果を比較できます。

## 主な設定

- Charm / Costume の選択
- 自己ステータス入力
- 敵の `DEF` / `Sp. DEF` 入力
- `AD` レベル設定
- Advanced Settings からの実験的オプション切り替え

## 入れ替えモード

入れ替えモードでは、一枚のメモリアを差し替えたときに全体の期待値へどの程度影響が出るかを確認できます。
